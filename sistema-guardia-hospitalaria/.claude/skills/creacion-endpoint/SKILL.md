---
name: creacion-endpoint
description: Crear o modificar endpoints FastAPI siguiendo las reglas del proyecto.
---

1. Identificar la operación solicitada y la entidad involucrada.
2. Determinar request, response y códigos HTTP correspondientes.
3. Verificar si existen schemas reutilizables. Crear o actualizar schemas si es necesario.
4. Verificar si existe un endpoint similar. Crear o actualizar routers según corresponda.
5. Aplicar validaciones y reglas de negocio definidas para el dominio.
6. Verificar consistencia con las entidades y estados definidas en el proyecto.
7. Verificar consistencia con el resto de la API REST.
8. Informar los cambios realizados.
