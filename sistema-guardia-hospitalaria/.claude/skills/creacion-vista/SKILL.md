---
name: creacion-vista
description: Crear nuevas páginas o componentes del frontend siguiendo las convenciones del proyecto.
---

1. Identificar el objetivo de la funcionalidad solicitada.
2. Determinar si se requiere crear:
   - una página en `pages/`
   - un componente reutilizable en `components/`
   - o ambos.
3. Identificar las necesidades de datos y las integraciones con el backend.
4. Reutilizar componentes, servicios y stores existentes cuando sea posible.
5. Si se requiere consumir información que no se encuentra disponible mediante la API actual, solicitar confirmación antes de crear nuevos endpoints.
6. Mantener consistencia visual y funcional con el resto de la aplicación.
