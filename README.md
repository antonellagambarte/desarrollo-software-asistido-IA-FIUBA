# Desarrollo de Software Asistido por IA - FIUBA

Repositorio utilizado para la materia **Introducción al desarrollo de software asistido por IA** de FIUBA.

Contiene ejercicios y proyectos desarrollados utilizando herramientas de asistencia basadas en inteligencia artificial para apoyar las distintas etapas del ciclo de desarrollo de software.

## Contenido

- Actividad 1: creación de bad UI con ChatGPT.
- Actividad 2: especificación de API REST utilizando estándar OpenAPI utilizando ChatGPT.
- Sistema de guardia hopitalaria: sistema para el manejo de guardia hospitalaria. Realizado con asistencia de CLAUDE.
- Presentación del proyecto de sistema de guardia hospitalaria.
